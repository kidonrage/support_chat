# Authentication FAQ

## I can't log in to NimbusPay

If you cannot log in, first check that you are using the same email address that is linked to your NimbusPay account. A common reason for failed login is attempting to sign in with a different email than the one originally used during registration.

NimbusPay uses one-time verification codes for login. If the code is expired, request a new one and use the most recent code only. Older codes become invalid as soon as a new code is issued.

## I did not receive the login code

If you did not receive the login code:

1. Check your spam or junk folder.
2. Wait up to 2 minutes before requesting a new code.
3. Make sure your email inbox is not full.
4. Verify that you are using the correct email address.

Repeated requests in a short period can trigger a temporary cooldown. During cooldown, new codes may be delayed.

## My code is invalid

The most common reasons for an invalid code are:

- the code has already expired
- a newer code was requested and the previous one became invalid
- the code was entered for a different email address
- the account email has not been verified yet

## I changed my email and cannot sign in

If you recently changed your email address, make sure the change was fully confirmed. Until email verification is completed, login attempts may fail or verification codes may be sent to the previous address.

## My account is active but login still fails

If your account status is active and login still fails, the issue is most often related to:

- expired one-time code
- unverified email change
- cooldown after too many code requests
- entering the code in a different browser session than the one where the login was started
