# Authentication troubleshooting

If a user cannot sign in after changing the phone number or device, first check whether multi-factor authentication still points to the old number.

Recommended steps:

1. Ask the user to try password reset and confirm the account email.
2. Check whether a recent device change or repeated failed sign-in attempts are visible in account signals.
3. If MFA delivery goes to an old phone number, route the user to manual identity verification or human support.
4. If the user can sign in on another device, ask them to refresh security settings and update the phone number from account settings.

Never promise that MFA can be disabled automatically. Identity-sensitive changes require verification.
